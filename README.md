# LeetCode 100

> LeetCode 精选 100 题刷题记录

## 项目简介

本项目用于记录和练习 LeetCode 精选 100 题，使用 Java 语言实现。

## 项目结构

```
leetcode100/
├── src/main/java/com/
│   ├── 双指针/
│   │   └── 两数之和/
│   ├── 哈希/
│   └── ...
├── pom.xml
└── README.md
```

## 技术栈

- **Java**: JDK 8
- **构建工具**: Maven

## 环境要求

- JDK 8 或更高版本
- Maven 3.6 或更高版本

## 编译运行

```bash
# 编译项目
mvn compile

# 运行项目
mvn exec:java -Dexec.mainClass="com.哈希.Main"
```
